Key Technical Decisions
設計上の主要な技術判断とトレードオフ
THE RAG
SQLite over PostgreSQL
デプロイ簡易性を優先。WALモードで同時読み取り対応。FTS5で全文検索も実現。小〜中規模チーム向け。
SSE over WebSocket
チャットは一方向ストリーミングのため SSE で十分。プロキシ互換性も高く、実装がシンプル。
Qdrant over Pinecone/Weaviate
オンプレ Docker 運用前提でセルフホスト可能。REST + gRPC 対応、永続ボリュームで堅牢。
Agentic Search (tool-use)
単純RAGでは回答困難な複雑クエリに対応。Claude tool-use で自律的にドキュメント探索。
Hybrid Search (BM25 + Vector)
セマンティック検索だけでは型番・品番の完全一致に弱い。FTS5のBM25を併用しカバー。
UUID-based Auth (No Login)
社内閉域網前提で認証レスに。LocalStorage UUID で個人設定を保持。導入コスト最小化。
